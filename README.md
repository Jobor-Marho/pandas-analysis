# Salary Analysis by College Major (Python & Pandas)

## Project Overview

This project explores salary outcomes for different college majors using Python and the Pandas library.
The goal is to analyze salary trends such as starting salaries, mid-career earnings, and salary distribution across major categories.

The dataset contains salary information for graduates across several fields including **Business, STEM, and Humanities (HASS)**.

---

## Objectives

The analysis answers key questions such as:

* Which college major has the **highest starting salary**?
* Which major has the **highest mid-career salary**?
* Which majors have the **largest salary spread** between low and high earners?
* Which majors offer the **highest earning potential** (90th percentile)?
* How do salary outcomes differ across **major groups (Business, STEM, HASS)**?

---

---

## Key Insights

**Highest Starting Salary**

* Engineering majors tend to offer the highest starting salaries.

**Highest Mid-Career Salary**

* Chemical Engineering and Computer Engineering rank among the highest.

**Highest Earning Potential**

* Economics and Finance show very high 90th percentile salaries, indicating strong upside potential.

**Salary Spread**

* Majors like Economics and Finance show large spreads, suggesting high variability in career outcomes.

**Group Analysis**

* STEM majors generally show higher average salary metrics compared to Business and HASS fields.
